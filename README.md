# 截圖拼接工具 · Screenshot Stitcher

把多張截圖一拖就拼成一張長圖。純瀏覽器執行，檔案不會離開你的電腦。

🔗 **線上版**：https://ayumiya90.github.io/screenshot-stitcher/

## 特色

- **純前端**：所有圖片在瀏覽器內處理，不上傳到任何伺服器
- **零安裝**：單一 HTML 檔，雙擊即開
- **多種輸入方式**：拖曳、點選、`Ctrl + V` 貼上剪貼簿截圖
- **拖曳排序**：縮圖直接拖動互換位置
- **對齊寬度四選**：以第一張為準 / 最窄 / 最寬 / 原圖不縮放
- **可調間距 & 背景色**：想留空、想換底色都行
- **即時預覽**：設定一改，預覽立刻更新
- **中文介面**、無廣告、無浮水印、無使用次數限制

## 使用方式

### 線上版
直接打開 <https://ayumiya90.github.io/screenshot-stitcher/>

### 離線使用
1. 下載 [index.html](./index.html)
2. 雙擊打開，沒網路也能用

## 操作流程

1. 把截圖拖進區塊（或點擊選檔、或 `Ctrl + V` 貼上）
2. 拖曳縮圖調整順序（右上 × 可移除單張）
3. 設定拼接方向、對齊方式、間距、背景色
4. 點「下載長圖」取得 PNG

## 技術

純 HTML + CSS + JavaScript，無任何外部依賴。使用 Canvas API 進行圖片合成。
